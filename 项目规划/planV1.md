# AI 费曼学习法 · 项目规划 V1

## 1. 项目目标

**目标用户：初中数学**（初一～初三，全学段覆盖）。

面向该人群打造一款用户定制化、区别于通用大模型的「费曼学习法」App（**平板端**运行）。

- **核心差异**：不是通用聊天，而是围绕「讲题 → 追问 → 掌握度更新」的闭环学习
- **商业假设**：家长用户会有更强的付费意愿（可查看学习进度、弱项、精彩讲题回放）

---

## 2. 考核要求（迭代导向）

> 不追求大而全，先做出经用户验证的原型。

| 原则 | 说明 |
|------|------|
| 快速迭代 | 多版迭代，基于反馈迅速出下一版 |
| 找真实用户 | 先有原型 → 用户验证 → 收集反馈 → 再改 |
| 聚焦核心需求 | 不必严格按文档限制；**需求本身最重要** |
| 克制范围 | 功能不做大而全，但**目录结构做全**；实现上先跑通**第十六章 二次根式**的最小闭环 |

---

## 3. 交互模式

| 输入 | 输出 |
|------|------|
| 手写轨迹 | Agent **音频**输出（同时展示**文字**，便于回看与检索） |
| 音频 | 同上 |

## 4. 产品形态 · 学生端

### 4.1 双端分离

- **学生端**：讲题、练习、与 Agent 互动
- **家长端**：实时/异步查看学习情况（详见第 5 节）

### 4.2 知识库与题库

- 按**章节 → 知识点 → 题型**切分，系统预置知识库与题库
- 用户可**上传题目**，或由 Agent **生成题目**
- 课本等内容**向量化索引**，Agent 具备检索工具（非整本 PDF 硬塞上下文）
- 选定知识点后，系统**推送对应题目**
- 用户**拍照上传**题目时，Agent 需能**识别所属知识点**

**目录 vs 内容（V1 策略）：**

| 层面 | V1 做法 |
|------|---------|
| **目录** | **做全**：覆盖初中数学完整章节树（初一～初三），用户可浏览全貌、感知产品边界 |
| **内容** | **只填「第十六章 二次根式」**（八年级下册）：16.1～16.3 共 3 节；题目、知识库、讲题流程、掌握度仅在该章落地 |
| **其余章节** | 目录中可见，但无题目/不可进入练习，统一标记为「即将上线」或置灰 |

> 原则：**壳子做全、肉先填一块**——让用户看到完整产品形态，研发资源集中打磨「二次根式」章节的体验。

**V1 上线章节**：八年级下册 · **第十六章 二次根式**（`pep-g8-down-ch16`，含 16.1 / 16.2 / 16.3）

**目录数据文件**：`data/curriculum/pep-junior-math.json`（6 册 · 29 章 · 90 节，含 `id` / `contentStatus` / `v1Launch`）；前端通过 `@/data/curriculum` 读取。

### 4.3 学习历史与掌握度

- 每个知识点下记录**练习历史**
- 每次练习后更新该知识点的**掌握等级**
- 掌握度影响后续：**出题难度**、**追问深度**（掌握越好 → 题越难、问越深）

### 4.4 多 Agent 讨论（费曼核心）

学生向多个具有不同「人设」的 Agent 讲解，例如：

| Agent 类型 | 特点 |
|------------|------|
| 基础不牢型 | 追问概念与前提 |
| 计算粗心型 | 抓步骤与计算细节 |
| 总结建议型 | 希望用户归纳、给出结构化反馈 |

- 各 Agent 按自身特点提问，**可在讨论中「吵起来」**（制造真实课堂感）
- **教师 Agent**：
  - 把控讨论方向与**结束时机**
  - 讨论结束后**总结**，并**生成变式题**
  - 学生卡住时给**间接提示**（不直接报答案）

### 4.5 步骤高亮

Agent 提问时，**高亮**题目/解题过程中对应的**相关步骤**。

### 4.6 今日悬赏

- 按**年级 + 学习进度**，每日推送 **3 道易错题**的错误解题过程
- 任务：找出错在哪，并**讲明白**为什么错

## 5. 产品形态 · 家长端

| 功能 | 说明 |
|------|------|
| 弱项看板 | **分维度**精确展示学生弱项（非单一分数） |
| 总结海报 | 一键生成可转发朋友圈的学习总结海报 |
| 精彩回放 | 查看学生**精彩讲题过程**（视频） |

---

## 6. V1 优先级（建议）

1. **必做**：初中数学**完整目录树**（全章节展示，仅「二次根式」有内容）
2. **必做**：**第十六章 二次根式**讲题闭环（手写 + 语音 + 多 Agent 讨论 + 教师总结）
3. **必做**：该章掌握度记录与难度调节（16.1 概念 / 16.2 乘除 / 16.3 加减）
4. **可选**：家长端弱项看板（简版，仅覆盖二次根式章）
5. **后续**：今日悬赏、海报、视频回放、逐章填充内容
